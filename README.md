# Juego del Ahorcado

## Descripción del Proyecto

Este proyecto consiste en el desarrollo de un juego interactivo del Ahorcado utilizando Python. El objetivo del juego es que el usuario adivine una palabra secreta ingresando letras una por una antes de quedarse sin intentos.

El sistema selecciona aleatoriamente una palabra de una lista predefinida y muestra al jugador la cantidad de letras que contiene mediante guiones. Cada vez que el usuario ingresa una letra, el programa verifica si pertenece a la palabra oculta. Si la letra es correcta, se revela en todas las posiciones correspondientes; si es incorrecta, se reduce el número de vidas disponibles.

Durante el desarrollo se aplican conceptos fundamentales de programación como estructuras condicionales, ciclos repetitivos, manejo de cadenas de texto, listas, validación de datos y generación aleatoria de información.

## Objetivos

* Implementar un juego funcional del Ahorcado.
* Aplicar lógica de programación mediante condicionales y bucles.
* Utilizar estructuras de datos para gestionar palabras y letras.
* Desarrollar una solución organizada siguiendo principios básicos de ingeniería de software.
